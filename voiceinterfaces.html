Research | Web Scraping | Data Analysis | Code

I explored classic research methods like in home interviews and secondary research to understand the practices, barriers, and attitudes of voice interface users. In an effort to increase the validity of my findings from the interviews, I built a simple web scraper with Python to gather more data from Reddit and did a sentiment cluster analysis.

I also explored how to build skills using Amazon Alexa and how to build Alexa into a Raspberry Pi and that it is very difficult to use.
